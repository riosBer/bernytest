# Block 2 — Practical Tutorial

This repository is used for the Block 2 practical tutorial.

**Note:** This repository is for coursework practice and is not graded.  
However, the skills and workflows practised here will be assessed in a later assignment.

The Block 2 tutorial will guide you through practising:

- GitHub Classroom
- GitHub Codespaces
- Running a simple web server
- Basic Git workflow (add → commit → push)
- Collaboration tools (Slack, Trello, Diagrams.net, Figma)

---

## What to do

Follow the Block 2 Tutorial document provided by your lecturer.

As you work through the tutorial, you will:

1. Open this repository in Codespaces.
2. Edit `index.html`.
3. Run a small web server.
4. Open the forwarded URL.
5. Make commits and push them to GitHub.
6. Upload your diagram export (if instructed).
